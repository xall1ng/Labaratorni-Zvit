# Лабораторна робота №1
## Завдання 1. Ввести одномірний масив А з N елементів та виконати наступні дії, змінити знак останнього елемента на протилежний.
# C++
```c++
#include <iostream>

using namespace std;

int main()
{
    int A[100];
    int n;
    int i;
    cout << "Введіть кількість елементів масиву: ";
    cin>>n;
    for(i=0; i<n; i++){
        A[i]=rand()%80;
        cout<<A[i]<<endl;
    }
    
    n = n - 1;
    
    A[n] = A[n]*-1;
    cout<<A[n];

    return 0;
}
```
